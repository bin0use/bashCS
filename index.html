Basic Terminal Shortcuts

CTRL L Clear the terminal
CTRL D Logout
SHIFT Page Up/Down Go up/down the terminal
CTRL A Cursor to start of line
CTRL E Cursor the end of line
CTRL U Delete left of the cursor
CTRL K Delete right of the cursor
CTRL W Delete word on the left
CTRL Y Paste (after CTRL U,K or W)
TAB Auto completion of file or command
CTRL R Reverse search history
!! Repeat last command

Basic Terminal Navigation

ls -a List all files and folders
ls List all files in folder
cd Change directory
cd / Go to root
cd .. Go up one folder
du -h Disk usage of folders, human readable
du -ah Disk usage of files and folders, human readable
pwd Print working directory
man Shows manual

Basic File Manipulation

cat Show content of file
mkdir Create a new folder
cp Copy and rename a file
cp / Copy to folder
mv / Move file in folder
mv Rename file
rm .. Delete file(s)
rm -i .. Ask for confirmation each file
rm -f Force deletion of a file
rm -f / Delete folder
touch Create or update a file
